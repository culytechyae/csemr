FIREWALL:
---------
If users cannot access the server:
- Ensure Windows Firewall allows port 5005
- Run this command as Administrator:
  New-NetFirewallRule -DisplayName "Taaleem EMR - Port 5005" -Direction Inbound -LocalPort 5005 -Protocol TCP -Action Allow

TROUBLESHOOTING:
---------------
- If you cannot access: Check if server is running (netstat -ano | findstr :5005)
- If connection refused: Check firewall settings
- If page not loading: Check server logs for errors

SERVER MANAGEMENT:
-----------------
Stop Server: 
  Get-Process -Name node | Where-Object {$_.MainWindowTitle -eq ''} | Stop-Process

Start Server:
  cd C:\EMR
